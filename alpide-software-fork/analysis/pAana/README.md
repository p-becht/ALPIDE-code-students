# pAana
Software of pALPIDEfs analysis

## pAAnalysis
- Analysis software

## pAnaScans
- Macros used to get scan info from the trees

## ReadScans
- Macros used to read scan data and convert into trees

## ChipScans
- Scripts used to setup/check pALPIDEfs chips and impliment the std scans
